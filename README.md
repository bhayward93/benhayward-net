# benhayward.net

This repository is for my personal site, hosted at [https://benhayward.net/](https://benhayward.net/).

## Installation

After cloning the repo and CDing into the directory

```sh
npm i
```

## Usage

```sh
# serve on port 3000 
npm run dev

# build
npm run build

# start
npm run start

# lint
npm run lint
```
